# React + Vite

# Project title
 ShopyGuys Application

## Description
 ShopyGuys application is a full-stack e-commerce web application developed using the MERN stack. The web application allow users to shop online. 

## Features
### For users
 - User Signup and Login
 - Update user profile
 - Add to cart and remove from cart functionality
 - Create order with delivery Adress 
 - View all orders
 - Apply Coupon
 - Payment page

### For Admin

 - Admin Home page
 - Create products (with Images)
 - Update Product
 - Create Coupons with Expiration

## Tech Stack
### Frontend

 - React
 - Vite
 - Redux Toolkit for state management
 - Redux-persist
 - React Hooks
 - JWT tokens(Authentication)
 - Axios
 - React Router
 - CSS
 - Vercel (Depoloyment)

## Environment Variables

Create a `.env` file in the root directory and add:

```env
VITE_BACKEND_URL=https://shopyguysapplications.vercel.app/api

```
## Installation

1. Clone the repository

```bash
git clone https://github.com/swathi-babashankar/shopyGirl-App-frontend.git
```

2. Navigate to the project directory

```bash
cd shopyGirl-App-frontend
```

3. Install dependencies

```bash
npm install
```

4. Create a `.env` file in the root directory and add:

```env
VITE_BACKEND_URL=https://shopyguysapplications.vercel.app/api
```

5. Start the development server

```bash
npm run dev
```
## Available Scripts
### `npm run dev`
 Runs the app locally

### `npm run build`
 Builds the app for production

### `npm run preview`
 Previews the production build locally
